但面对几千只美股，怎么真正**落地执行**？

我用 Python 把他的理论**变成了代码** ——

**Graham Investor**：一个格雷厄姆 + 巴菲特混合策略的美股量化选股系统。

---

# 📐 核心：Graham Number 公式

> Graham Number = √(22.5 × EPS × BVPS)

这是格雷厄姆估算股票**内在价值**的经典公式。

市价低于这个数 → 存在**安全边际** → 可能被低估。

但光看一个公式远远不够 👇

---

# 🎯 12 维加权评分体系

不靠单一指标，从**四大维度**综合打分：

**估值（30%）** → P/E、P/B、Graham Number 安全边际

**财务（15%）** → 流动比率、债务权益比

**成长（28%）** → 连续盈利年数、分红年数、盈利增长

**质量（27%）** → ROE、净利润率、自由现金流

五级评级：**A ≥ 80 强推** → B → C → D → **F < 35 不推荐**

# 🛡️ 防价值陷阱机制

最怕的就是"看起来便宜，其实在衰退"。

系统内置**盈利衰退惩罚**：

盈利增长为负时，总分乘以 **0.45 ~ 0.80** 的惩罚系数。

便宜但在走下坡路的股票？直接降级 ❌

# 🚀 一行命令扫全美股

**两阶段流水线**自动运行：

① NYSE + NASDAQ 粗筛 → 200 只候选

② 逐只精细评分 → 输出完整排名

```
python -m graham_investor.model --top10
```

还支持指定分析：

```
python -m graham_investor.model AAPL MSFT GOOGL
```
